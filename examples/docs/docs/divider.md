## Divider 分割线

### 按需引入

```javascript
import Vue from 'vue'
import { Divider } from 'wot-design'

Vue.use(Divider)
```

### 基本使用

使用默认插槽设置内容。

```html
<wd-divider></wd-divider>
```

### Attributes

| 参数 | 说明 | 类型 | 可选值 | 默认值 | 最低版本 |
|-----|------|-----|-------|-------|---------|
| color | 自定义颜色，所有颜色的写法 | string | - | - | - |

### Slot

| name | 说明 | 最低版本 |
|------|-----|---------|
| default | 内容 | - |
